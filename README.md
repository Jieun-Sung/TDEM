# TDEM

## Installation

1. Clone the repository and download the `TDEM.yaml` environment file.
2. Create the conda environment:
    ```bash
    conda env create -f TDEM.yaml
    ```
3. Activate the environment:
    ```bash
    conda activate TDEM
    ```
